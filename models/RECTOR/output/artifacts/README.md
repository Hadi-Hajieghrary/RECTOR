# RECTOR Publication Artifacts

This directory contains all publication-ready figures and LaTeX tables for the RECTOR paper. Every artifact is generated programmatically from evaluation results — no manual editing or post-processing is applied.

The core principle: **every number in every table and every data point in every figure traces back to `canonical_results.json`** (or the training logs), ensuring consistency across the paper.

## Regenerating All Artifacts

```bash
cd /workspace/models/RECTOR/scripts
python artifacts/generate_all_artifacts.py
```

This master script runs all individual generators sequentially, captures errors, and writes `summary.json` with generation status and timing for each artifact.

Individual scripts can also be run independently:

```bash
python artifacts/generate_training_curves.py
python artifacts/generate_rule_violation_analysis.py
python artifacts/generate_ablation_study.py
# ... etc.
```
