# RECTOR Utilities

This directory contains shared helper code that supports the rest of the project. The most important utilities are related to visualization and reporting.

## Main role

Utilities in this directory are not usually the focus of the scientific contribution, but they matter for:

- consistent visual styling,
- reusable rendering logic,
- and keeping the main experiment scripts concise.

## Typical contents

The most visible helper in this directory is the visualization support used during training and qualitative inspection.

## Recommendation

Treat this directory as supporting infrastructure. It is helpful when reproducing figures or training dashboards, but it is usually not necessary to describe it in detail in the paper unless a visualization design itself becomes part of the contribution.
